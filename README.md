# fn-ui-avatars-docs

[![Vercel Deployment](https://therealsujitk-vercel-badge.vercel.app/?app=fn-ui-avatars-docs)](https://fn-ui-avatars-docs.vercel.app/)

The official repository for the documentation and interactive playground of the **fn-ui-avatars** ecosystem.

Here you can find the code for the Next.js application that serves as the live demo, allowing developers to test the deterministic avatar generation and the smart contrast algorithm in real time.

> **✨ Live Demo:** [Try the interactive generator here!](https://fn-ui-avatars-docs.vercel.app/)

---

## 🚀 Built With

This application was developed with a focus on performance and a smooth user experience:

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Components:** [React](https://react.dev/) & [fn-ui-avatars-react](https://github.com/fnasciment0/fn-ui-avatars-react)
- **Style:** [Tailwind CSS](https://tailwindcss.com/)
- **Hosting:** [Vercel](https://vercel.com/)

## 🛠️ Local Development

If you want to run the playground on your local machine to test changes or contribute to the public interface, follow the steps below:

### 1. Clone the repository

```bash
git clone https://github.com/fnasciment0/fn-ui-avatars-docs.git
cd fn-ui-avatars-docs
```

### 2. Install dependencies

```bash
npm install
# or
yarn install
# or
pnpm install
```

### 3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```
Open your browser at [http://localhost:3000](http://localhost:3000) to see the result.
